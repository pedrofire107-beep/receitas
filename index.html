Sobremesas

Torta de Amêndoas

Ingredientes (1 / 2 / 4 pessoas):

  • Amêndoas: 20g / 40g / 80g
  • Farinha: 50g / 100g / 200g
  • Açúcar: 15g / 30g / 60g
  • Ovo: 1 / 2 / 4

Modo de preparo:

  1. Misture farinha, açúcar, ovos e amêndoas.
  2. Asse 180°C por 25 min.

Molho Branco (Bechamel)

Ingredientes (1 / 2 / 4 pessoas):

  • Manteiga: 10g / 20g / 40g
  • Farinha de trigo: 10g / 20g / 40g
  • Leite: 50ml / 100ml / 200ml
  • Noz-moscada: 1 pitada / 2 pitadas / 4 pitadas

Modo de preparo:

  1. Derreta manteiga, adicione farinha e cozinhe 2 min.
  2. Acrescente leite aos poucos, mexendo até engrossar, finalize com noz-moscada.
